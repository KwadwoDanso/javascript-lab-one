##Reflections

-How did you approach creating more flexible functions with parameters? Making the parameters specific. Making sure that the parameters are carefully handled in the flow. 

-What challenges did you face while incorporating conditional logic in your functions? Checking all of the cases. Being mindful of over 18  and older than or equal to 18 of age. 

-How does refactoring improve the readability and maintainability of your code? Before refactoring I needed to separate the functions or duplicate the logic to handle the discounted price. Readability improves with intuitive function names and clear parameters make it easy for people to know the direction of the function. It is more maintainabe because it is isolated and modularized so it can be easier ro fix if needed.
